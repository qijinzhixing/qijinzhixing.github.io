刻意练习

半年前了解到了“刻意练习”这一个概念，平时自己也会做一些练习，但是有时候练习的效果不是那么的理想，所以这里把这么重要的方法做一个整理和回顾，以加深对于“刻意练习”的理解。

一、刻意练习是什么？

刻意练习是为了提高绩效而被设计出来的一种练习方式。最早由佛罗里达州立大学的心理学家 K. Anders Ericsson提出。刻意练习的假设是，世界级的水平是逐步练习得来的，关键就在于让受训者完成一些自己不会但是通过学习就可以掌握的练习，要求受训者在这个过程中高度集中注意力。

二、刻意练习的生理依据是什么？

刻意练习和一种叫做“髓鞘质”的物质密切相关。人的任何一个技能，都是大脑内一系列神经纤维传递的电脉冲信号的组合。任何时候，你做的每一个动作，大脑就发出一个信号，通过神经纤维传达到你的肌肉。而髓鞘质的作用就像绝缘胶布那样包裹着这些神经纤维，以防止电流脉冲外泄。这就意味着，髓鞘质把神经纤维包裹的越好，电流脉冲外泄越少，你的技能就会越精确。如此可见髓鞘质对于刻意练习的重要性。借用加州大学洛杉矶分校的神经学教授乔治·巴特克斯（GeorgeBartzokis）博士的话，髓鞘质是“交流、阅读、学习技能、人之成为人的关键”。

三、刻意练习有哪些特点？

1. 只在“学习区”练习

心理学家把知识和技能分为层层嵌套的三个圆形区域：舒适区、学习区和恐慌区。舒适区是那些我们很容易就做到的事情，一直呆在里面的话很难有提高。恐慌区是我们暂时还没有掌握的区域，我们对于这一区域非常陌生。中间是学习区，也就是那些通过学习或练习就可以掌握的知识和技能。
刻意练习要求练习者在学习区域进行练习。国内有一个名词：因材施教。和这里表达的意思一致。也就是根据练习者当下的能力水平，制定合理的练习任务。这些任务是动态的，如果某些技能已经练习到了非常舒适的程度，那么就要转入下一个难度的练习。针对性非常强，而不是没有目的的进行练习。

2. 大量重复练习

刻意练习的重中之重是大量重复。大量重复可以帮助你从不会到熟练的掌握某些技能。比如篮球运动员大量练习投篮，将一个简单的罚球动作重复练习数以万次，或者是重复练习某个距离的投篮；再如钢琴演奏家大量练习钢琴，针对某一段音乐反复练习。这些都是重复练习。伟大的成功者之所以伟大，在于重复练习的量基本都达到了单调乏味的程度。

3. 持续获得有效的反馈

如果一项练习没有获得效果，结果会是，要么你不再取得进步，要么你不再在乎了。这就是说想要取得进步，获得持续的效果反馈是非常有必要的。对于有些活动，例如体育或竞技比赛、象棋等，获得反馈是很容易的，看看比赛的结果就知道了。但是如果结果需要解释，那么问题就来了。举例来说，你认为你面试准备的十分充分，但是结果是你没有获得你想要的职位。怎么解释呢？难道面试官不识人才？此时你自己的反馈意见是不是可靠的呢？不见得！这种情况下，需要把问题交个专业人士去做，可以的话，面试完了询问一下HR的真实想法，为什么不录用你？自己的那些地方展现的不好？专业人士的反馈意见是需要重视的，因为他们往往能够看到自己看不到的地方。这就是老师、教练或者导师起到的关键作用。
引申一下，不管是参加面试，还是体育比赛，或是其他的练习，找到一个该方面能力强的人指导一二，这对个人的成长是极其有帮助的。不过获得有效反馈的最高境界是自己给自己当教练。这一点非常难以做到，也就是练习的时候，确定一个具体的小目标，以一个旁观者的身份去观察自己的练习，敏锐的发现自己的一些错误，然后寻求改进。《学习之道》的作者乔希·维茨金就是一位善于给自己当教练的人，所以他能够在象棋和推手两个看似毫无关联的领域都取得世界冠军。即便是乔希这样的高手，他依旧需要教练，持续不断的给予有效的反馈。

4. 精神需要高度集中

刻意练习不同于普通的练习，它是要找出那些动作和绩效上做得不够满意的某些元素，然后努力去改善它们。这个过程需要精神高度集中，对于个人的专注能力提出了一个很高的要求，如果专注能力较弱的人，很难做到“刻意”。这就是为什么普通练习进步慢的一个原因。
关于刻意练习的练习时间研究有个最显著的发现，一天内刻意练习的时间上限是4-5小时。但是人的精力十分有限，无法做到连续几个小时内保持高度集中。所以即使按照一万小时天才理论每天练习三个小时，也需要分成两三段去完成。一来你不可能连续三个小时内都保持思想高度集中，二来如果思想高度集中的进行练习，一个半小时就可达到全天不停地普通练习同样的效果。

5. 没有多少乐趣可言

刻意练习没有乐趣是显而易见的。它是离开我们的舒适区进行练习，找到那些我们并不擅长的地方进行改进。这个过程需要重复那些让我们感觉不舒服甚至痛苦的练习。这一点是通往天才的毕竟之路，没有办法回避，所以就过程来看，它会使人身心俱疲。这就是我们看到的，对一些有成就的运动员进行采访的时候，很多运动员表示不喜欢自己擅长的。是不喜欢获得奖项和荣誉吗？不！这其实就是不喜欢令人不舒服的刻意练习过程。可见刻意练习的过程多么无趣。
因此，想要获得世界级的水平，要么对于自己所练习的东西有真正狂热的激情（通常较少人是这样的），用狂热的兴趣抵消对痛苦的体验；要么就做好强大的心理准备接受它。

四、刻意练习究竟如何练习？

一个人选择培养的能力和技巧是无限多的，但实践它们的机会可以分为两种：一种是直接练习的机会，像音乐家在演出前练习某个乐章那样。第二种是作为工作的一部分去练习。下面分别展开说明。

第一种方式：直接练习

根据刻意练习在不同领域中所起作用的关键程度，把可以练习分成三个大的不同种类：音乐模式、国际象棋模式和体育模式。

音乐模式：你知道自己想要传达什么信息，但挑战在于如何才能高效地将其传达出来。信息可以被拆分成小片，每一小片都进行意图分析，然后反复练习，由教练或者录像提供即时反馈。一首具体乐曲的的演奏内容是固定不变的，如贝多芬的《月光奏鸣曲 》（moonlight sonata ) ，这种练习模式的做法就是把这首《月光奏鸣曲 》（moonlight sonata ) 拆解成很多段，然后分别针对于某一段进行练习。音乐模式可以适用在一些内容固定的场合，如某个固定主题的演讲，某项重复性的工作，这里可以是练习双拼输入法，甚至可以是戴维·艾伦的GTD系统。

国际象棋模式：这和商业中的案例分析法几乎是一个概念。你面对一个问题，而且你需要专注于问题和评估多种解决办法，并找到一种解决方法。它的精髓是：看到外部的变动，你将会做什么？比如象棋，研究某个特定的位置，然后做出自己的选择，与大师的选择进行比较，如果结果不同，那就分析一下哪种方法更优。国际象棋模式可以应用在案例分析的练习上面，具体的如练习结构化的思考能力等。

体育模式：优秀运动员的训练分为两类：体能训练和特殊技能训练。它和音乐模式最大的区别在于，面对的情况不会完全一样，做出的动作也会有所不同。
体能训练意味着你要着重回顾你的基础技能知识，比如word、Excel等的使用技巧，以便于更快更有信心的使用它们。基础的技能知识可以是底层的专业知识，比如金融和会计工作的基础数学能力，工程行业的基础科学能力，编程工作中的基础语言能力。
特殊技能训练是建立在高度刺激的基础上。和商业社会相似，不可预知的对手，快速的反应，不断变化的形势。这中练习需要遵从这样的一些原则：尝试提高自己技能的某个方面，大量重复，及时反馈。可以的话，最好找一个人配合你练习，比如谈判能力等等。

第二种方式：在工作中练习

直接练习的机会很多，但是也并非唯一的方式。在工作中可以有不同的方式去练习商业技能，在工作中的非直接练习称为自我调节。纽约城市大学的巴里·J·齐默曼教授指出，“刻意练习的一些要素被当做自我调节的主要成分而被研究”。有效的自我调节是一些你在工作以前、工作当中、工作之后所做的事情。

工作前：首先确定目标，这个目标不是巨大的人生目标，而是确定你当天将会做什么事情的目标；然后制定一个最为详细、最具技术导向的计划，表现最好人的制定的计划不只是专注结果，还专注于取得结果的过程。比如，不仅仅只是为了赢得合同，还要特别留意去发现顾客未说出的需求。这个过程需要我们有积极的态度和足够的信心，坚信自己能够完成，因为坚信自己的付出会有回报。

工作中：工作中最为重要的技能是自我观察。表现杰出的人士可以做到离开自己，监测自己大脑里发生的事情，并且询问事情进展如何。如提出这些问题问题：在这个情形中，什么能力受到考验呢？我能否尝试其他技巧呢？我能把自己推得更前一些吗？他做得如何？也就是说，杰出的人士可以一边做一件事情，同时练习另外一件事情。比如普通的长跑运动员在跑步的过程中往往转移自己的注意力，而长跑好手恰相反，他们强烈专注于自己所做的事情，数自己的呼吸和步伐，同时还做一些其他必要的事情来保持自身的协调统一。

工作后：一定要有结果的反馈，换言之就是进行自我评估，评价一下结果如何，关键在于找出错误的原因。通常优秀人士是利用自己正在努力达成的目标作为评判的标准，或者把自己的表现和状态最好时候相比较，看到差距，找出造成差距的原因，然后进行改进。当然，这个标准必须适中。它与刻意练习一样，必须要选择一种比较方法才能把你刚好拉出现有的局限之外。太高的标准会使人气馁且没有教育性，而太低的标准则无法使人进步。

五、结论

刻意练习是一个巨大的投资，需要你付出极多的时间和精力，但是通过刻意练习能极大程度提高自己的知识和技能这一结论毋庸置疑。如果你的目标只是练习到一半或者一般略偏上的水平，那么你可以通过辛苦一段时间就可以实现，比如三周内上手双拼超过之前的水平。但是如果你的想法是通过刻意练习达到世界级的水平，那么在进行刻意练习之前，有两个极为重要的问题必须回答清楚：你真正想要什么？你真正相信什么？
