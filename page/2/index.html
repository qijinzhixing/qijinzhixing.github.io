与Vim的第一次邂逅

By QJzhixing

发表于 2015-06-26

　　之前有在同学的推荐下使用了Notepad++这一款软件，那会对于文本编辑类的软件还没有任何的概念，用了一段时间，感觉对于效率的提升没有多大的作用，也只是用一下其中的替换、宏命令和消除格式作用。奇怪的是，在文本编辑器中，小众软件里它是排在前面的，但我没有发现它有太多特别之处。邂逅了Vim之后，Notepad++被直接弃之。原因很简单 ，对于高效编辑和排版有需求，但是Notepad++已经无法满足目前的需求。
　　对于刚接触Vim的人来说，上手确实需要耗费一定的时间和精力，尤其可能会被一些较长的命令串吓到，其实大可不必，长的命令串也只是一些简单的命令的组合而已，明白含义多用几次之后就习惯了。但不得不说，Vim的功能确实强大，花一点时间去学习和了解对于提高自己的效率十分有必要。

一、常用模式

　　一般而言，Vim的初始界面是Normal模式。在Noraml模式下，输入不同的命令进入不同的模式。按Esc键返回Normal模式。这是和其他普通编辑器区别很大的地方。以Notepad++来说，它只有一种模式——编辑模式，即输入什么就得到什么，比如输入“V”就显示“V”。而对于Vim来说，输入“V”不是得到“V”这一个字母，而是进入了Visual模式里面，在这个模式里面可以整行整块的编辑文本。再如输入J也不是得到J，而是合并行。这是模式之间的区别。除了这一模式，下来还有两种常用的模式：
　　: （英文冒号）命令模式
　　冒号之后输入不同的字母表示不同的命令，比如:w表示保存，又比如:X（大写）表示给文件加密，两次输入密码后即可给文件设定一个密码，若是TXT文件用Vim设定了密码，那么用自带的文本编辑器或者Notepad++打开就只能显示乱码，数字和字母也不正常显示，非常安全。
　　a（A）、i（I）、o（O） 进入编辑模式
　　进入编辑模式的方法有多种，这里主要指插入，大小写的字母分别代表了不同的含义：
　　a 在光标之后插入文本
　　A 在光标段末插入文本
　　i 在光标之前插入文本
　　I 在光标段首插入文本
　　o 在光标下面插入新行
　　O 在光标上面插入新行

二、基本操作

　1. 移动

　　在Normal模式下移动光标可以通过方向键进行操作，但更常用的是h、j、k、l四个按键，因为这几个键分布在右手常活动的区域。右手的食指、中指和无名指就可以搞定。这几个按键分别表示：
　　h 左；j 下；k 上；l 右
　　说明：后面的多数命令支持命令计数，命令计数指的是，在命令前面加上数值的话可以一次性进行多步操作。比如输入5j光标就向下移动5行，当然也可以通过输入jjjjj来实现的。

　2. 删除

　　删除字符（汉字），输入x即删除光标对应的字符。如果想要删除3个字符，可以输入xxx，也可以输入3x。
　　删除整行，输入dd即删除光标所在的行，同样的想要删除多行可以前面加上行数的数值。

　3. 撤销和对撤销的撤销

　　在编辑的时候，如果误删除了一些内容，可以通过u进行撤销，这个有点类似于Office里面的Ctrl+Z的作用，但如果撤销多了也不要紧，还可以通过Ctrl+r进行反撤销，不用担心因操作失误而导致数据的丢失。

　4. 改写

　　如果是对于其中的几个字符（或汉字）进行修改，那么直接用改写就很方便了。可以输入cw改写一个单词，也可以输入c3l改写3个汉字。另外，输入R进入替换模式，输入的每一个字符将替换掉光标上的一个字符。输入r是改写单个的字符，如果前面加上数值n，则表示输入n个同样的字符。

　5. 复制和粘贴

　　y 复制字符或单词，比如y3l就表示复制了3个汉字（对于英文则是3个字母），英文一个单词一个单词的复制是y3w
　　yy和Y 都表示复制了一整行
　　y$ 复制当前光标到行尾
　　p 基本充当的是粘贴的作用，对于用y复制的内容都可以粘贴到目标位置

　6. 统计字数

　　gCTRL-G，g后面没有空格，后面的不分大小写

　7. 其他操作

　　ZZ 表示退出
　　.（英文句号）表示重复上一次的动作

三、替换

　　替换之所以单独拿出来，是因为它的作用远不仅仅局限于查找和替换，还有一次性删除目标字符串，包括空格、空行（\r）等等。
　　输入命令行:%s/a/b/gc
　　% 表示命令将作用在所有行上面，如果不输入%则命令只作用在当前一行上
　　s 全称是substitute，一般缩写成s就可以
　　s/a/b 表示把a替换成b
　　g 表示一行中所有符合目标的字符串都会改变
　　c 表示确认，请求用户确认
　　输入上面的命令后得到下面的信息
　　替换为 b (y/n/a/q/l/^E/^Y)?
　　帮助里面的解释很详细，这里直接复制过来
　　y 好吧，替换吧
　　n 不，这个先留着
　　a 别问了，全部替换吧(这个比较常用)
　　q 退出，剩下的也不要管了
　　l 把现在这个改完就退出吧
　　CTRL-E 向上滚屏一行
　　CTRL-Y 向下滚屏一行
　　说明：1.如果b不输入即表示删除符合目标的字符串；2.也可以指定范围替换，前面的%实质上是等于1,$的命令，即替换范围是从首至尾，或者不用%，改用1,10或10指定前面10行替换都是可以的。

四、宏命令

　　使用Markdown语法的时候，需要在一级标题前添加一个#，然后习惯在一级标题下添加一行空行，尝试了一下宏命令，可以实现：

　　按Esc键返回到Normal模式，按^将光标置于行首
　　输入qa（q表示开始录制宏命令，a表示宏命令的名字，名字只能是a-z中间的一个字母）
　　再按一下i（在光标之前插入文本），输入#，按Esc返回
　　再按o（在下面插入一新空行），按Esc返回
　　再按j^（把光标移动到下一行首）
　　最后按q（结束宏命令的录制）
　　使用宏命令的话，把光标移到目标位置，输入@a就可以了。如果是前面已经确定好了一级标题的框架，那么在其前面添加相应的数值就可以实现。比如5@a，就可以一次性把本文的以及标题确定好。

五、列操作

　　列操作（实际上是进入Visual模式进行整行整块处理）在编辑排版非常高效，比如中文习惯于段首空两格，使用Markdown语法的时候需要在段首添加两个全角的空格才可以，如果一段段添加任务太过繁琐，使用列操作就方便很多。另外在一些行前面添加无序编号也非常的方便。这里以在段首添加两个全角空格为例：


　　将光标置于行首，按下Ctrl-q
　　按jjj……（选择需要改变的部分）
　　按I在行首插入，然后输入两个全角的空格
　　按Esc退出即可执行完毕

六、Vim乱码设置

　　有时候用Vim在打开一些文件之后会出现乱码的情况，其实是Vim编码设置的问题。主要有四种：encoding（Vim的内部编码）、termencoding（Vim在于屏幕/键盘交互时使用的编码）、fileencoding（Vim当前编辑的文件在存储时的编码）、fileencodings（Vim打开文件时的尝试使用的编码）。解决办法是在_vimrc文件里面按照需求添加相应的代码即可

　1. 文本乱码式

　　set encoding=utf-8
　　set termencoding=utf-8
　　set fileencoding=utf-8
　　set fileencodings=ucs-bom,utf-8,chinese,cp936

　2. vim菜单乱码

　　source VIMRUNTIME/delmenu.vim　　sourceVIMRUNTIME/menu.vim

　3. vim提示信息乱码

　　language messages zh_CN.utf-8

七、总结

　　知道Vim这一款软件也有一段时间了，但是长久以来对其不了解，迟迟没有投入自己的时间和精力去学习，一来是畏难心理的作用，二来是感觉没有需要，所以就误以为自己没有兴趣。现在看来，上手没有想象中的那么难，花一点时间学习使用一下就好了，不必追求技巧多么花哨，满足自己的需求就可以。即使不编程，单写作之后的编辑和排版就可以发挥其极高的效率。

10Km长跑，你也可以做到！

By QJzhixing

发表于 2015-06-14

先来看一看我的结果，11.8Km，耗时1h13m25s。这样的数据内行人看来真的不算什么，但和自己比，已经进步很多了，回顾自己的过往就会发现，这也是一个具有突破性的进展。


　　长跑对于我来说，可谓是不敢想象的事情，而这也今年非常大的一个意外收获。
　　一直以来，1Km都是一个巨大的挑战，更别说什么10Km，搁在一个月之前，都没有想过要跑那么远。原因在潜意识里，认定这根本就是一件不可能办到的事情。之所以认定不可能，和自己的经历有很大的关系，主要体现在两点上面：肺活量太差和跑不到100米就跑不动了。
　　肺活量到底有多差呢？在大学生体能测试中，我巨怕一种测试：测试肺活量。根据《国家学生体质健康标准》，对于男生而言，肺活量及格的最低标准是3100，而我费劲体力尝试突破3000都没有成功过，一直徘徊在2500~2600之间，这之间的差距太大了，非常令人泄气的一件事情。当然，这件事说来有些惭愧，为了应付考试拿到这项的及格分，最后是请同学帮忙替考了。这样的测试使得自己形成了一种固定的认知，即认定了“自己的体能非常差”的这一事实，而这样的固定的认知一直限制了自己。主要就体现在，对于一件新鲜的事物，自己还没有开始去尝试，就首先就把自己否定了。所以一直没有做出一件与自己来说像样的事情。
　　100米究竟有多远？对于运动员来说，也就10秒左右的样子。我尝试了多次跑100米，但每次的结果都大致相同，跑到70米左右的样子腿就迈不开了。且抛开100赛跑这样高强度的跑步，说说慢跑100米吧，慢跑的结果也是没有例外的，感觉自己就是迈步过去了，好像一说起跑步这项运动，压根就和自己是完全对立的。
　　更年轻些的时候，每天下午都要和同学一起去操场打篮球，狂热的打，不管刮风和下雨，基本少有例外的。打篮球也是一项运动啊，打了那么多年的篮球，但好像对增加肺活量没有太大的作用（后来才知道有有氧运动和无氧运动之分）。有时候尝试去跑步，即使咬牙坚持，但最远也不过两三百米。有了这些体验，所以跑步这个念头就一直被无情扼杀，几年就这样过去了，再也没有提起跑步这件事情。
　　去年11月份的时候，偶然看到了战隼老师的#100天行动计划#这样一个内容，感觉应该会非常的有用。想到自己每天睡得晚，早上又是各种的起不来，因为又年末了，天气特别的冷，经常是赖床，期间浪费了不少的时间。考虑到自己的论文需要着手撰写，需要大量的时间和充沛的精力，所以把战隼老师的行动计划做了一个调整，变成#60天行动计划#，主要的任务就是六点半点起床，接着出门跑几百米。头三天里，记录里出现的话语是：大小腿很酸痛，咬牙坚持。后面一直也是仅凭意志力去坚持，最后到元月29号的时候计划结束。结束之后，也做了一次总结，除了大小腿不再酸痛之外，好像也没有多大的进步，而且对于跑步这件事情的认知上也没有本质的改变，依旧感觉是一件痛苦的事情。
　　开年来，围绕论文展开的事情越来越多，也变得比以前更忙。在3月中旬的时候，为了保证论文顺利完成，再次启动了100天行动计划。这一次吸取了上次的一些经验，在制定规则的时候就没有那么刻板了，更详细考虑到一些意外的天气和重要突发的事情。行动计划的内容依旧是早起和跑步，不同的是，跑步的距离增加了一点（但也不超过1Km），持续的时间更长些。一直坚持了下来，中间也偶有一两天的中断，也基本没有什么问题。到了5月21号，应朋友之约，外出跑步，恰好他属于长期锻炼、体能极佳的一类人。我之前的跑步一直都是在校园里面绕圈子，第一次尝试和他出去跑步，也不能跑个几百米就不跑了，那太扫他的兴了，所以一路聊天一路坚持，感觉跑了很远的样子。到停下来的时候，两腿飘飘的，有点不像是自己的了。但我知道已经突破了，至少有两三公里的样子，回来记录了一下，4.6Km，全程耗费了52m08s。
　　有了这样的突破，我体会到了这样的一句话：“不要被你不知道的或没做过的事情所限制！”之前是明白这句话的含义的，但是和现在自己在实践中去领会这句话的是完全不一样的感受的。超过1Km以上的长距离的跑步，自己以前是没有做过的，所以限制了自己通过练习可以实现的想法，很大程度上阻碍了自己的进步。
　　这次下来，我开始陆续增大跑步的距离，抓着朋友去陪自己跑步。从4.6Km增加到了6.8Km，在从6.8Km增加到了8Km，这期间都是和朋友一起去跑，没有自己单独跑过。然后我意识到，后面参加工作了，想要跑步又没有跑伴怎么办呢？所以6月10号的时候，我开始尝试一个人去跑，我想看看自己能不能在没有人鼓励的情况下跑出较长的距离，也就是自己鼓励自己。最后真的做到了，9.2公里，耗时57m17s。因为想要突破10Km，所以休息了两天之后，6月12号再次尝试，一路听着自己整理的笔记，再次加大了跑步的距离。我想上面的结果我已经非常的满意了，当然不是满足，因为可能还可以做的更好。
　　我究竟是如何做到的呢？这样的突破性进展，如果不回头看看自己究竟是如何做到的，不把过程弄明白，那么以后也就很难再有其他的突破。
　从这件事情里，至少能学习到三点：

  • 首先，方法和技巧非常有必要。拿自己的经历来说，跑步的姿势，是入门要点。自己之前的跑步的姿势几乎完全是错误的，错在哪里呢？有两个地方非常的明显：其一，跑步身体不是直立的，一直前倾的厉害，然后跑到后程的时候，体能消耗非常大，慢慢的腰也开始变得不直立了，整个人似乎都是蜷着身体在跑，不累才怪；其二，头部不是固定在中间不动的，跑的时候有点摇头晃脑的，这也是非常忌讳的，头部摆动造成身体的不协调，也会增加体能的消耗；其三，手臂的摆动不协调，之前自己的摆动略有左右摆动的倾向，真正的做法应该是，手臂前后摆动，摆动的幅度不能过大，控制在自己的身体的腰部的位置；其四，腿部太过僵硬，之前一直是直起直落的姿态，也就是身体的高度在不断的变化，这样的结果就相当于每跑一步是跑步的基础上再做了一次蹦高，不用说，这样的做法更是消耗体能，现在纠正了这些，腿部自然起自然落，抬起的时候靠近自己的身体。
  • 其次，目标分解和跟踪改进。很早之前就知道日本一名跑者，每次在比赛前都要去熟悉场地，然后把整个路线设定N个小的段，一段一段的跑，结果非常有毅力且出色的完成了比赛。我想，这就是目标的分解，把跑步这个大的目标分解成一个个小一点、容易实现和达成的目标，整个过程坚持下来就容易完成了。在自己单独跑步的时候（路线较熟悉了），自己采用了这个简单的小策略，真的比较管用。第二个比较重要的，涉及目标的跟踪和改进，跑步计划坚持了两次，很长时间都没有什么突破型的进展，原因有一点，那就是一直在维持之前的水平，不去提高自己的目标，一直在原有距离里面跑，这是自己之前没有意识到的问题。也就意味着，想要有突破，那就是对自己的目标进行跟踪和不断的改进，不是一直维持在之前的水平，而是在维持一段时间之后，增大自己目标的难度，只有这样，才可能有持续不断的突破。真正的进步，我想，应该是螺旋上升的。

  • 最后，最重要的当然是心理的障碍，也就是上面的那句话。长久以来，我把自己能够做到这样的事情否定了。潜意识里的想法，自己很难去改变，潜意识里认为自己这件事自己办不到，然后自己就不去尝试，结果只能是真的办不到。这让我意识到，一件事情没有尝试过就不要急于否定。引申一下，对于新鲜的事物，在没有了解之前，先不要急于去否定。生活是非常有意义的，尽管自己目前对于这样的话体会不深，对于生活的意义领悟不够，但有一点，那就是有趣生活的本质就是不断的去尝试新鲜事物。

《如何高效撰写毕业论文》之一，Word技巧篇02

By QJzhixing

发表于 2015-05-29

一、如何插入分节符？

分节符，顾名思义，就是将章节与章节之间分隔开来。也就意味着，在调整其中某一章节之后，即使页数有所变化，后面的章节不受其影响。操作方法如下：

  1. 将光标置于章节的尾部，选择页面布局，分隔符，下一页，就可以插入。

二、如何在尾注后面添加学术成果和致谢？

以尾注插入参考文献后，参考文献会在文档的末尾显示，而参考文献后面还有学术成果和致谢等内容，直接在后面添加是不能实现的，需要通过以下的方式解决：

1.以尾注方式正常插入文献，正常分节；

2.把尾注设置成节的结尾，文献会分别显示在每个章节的后面；

3.把光标置于第1章的任意位置，选择页面布局，页面设置，勾选取消尾注，确定。这时，第1章的参考文献会自动合并到第2章的后面；

4.重复3操作，直到参考文献这一章节即可。
说明：操作到这一步先不要急于输入后面的学术成果和致谢部分，需要对参考文献这一章节进行分节，分节后在插入。对参考文献进行分节的方法请参考Word2003中参考文献后如何添加分节符。word2007版的类似。

三、如何正确设置页眉页脚？

学位论文需要添加页眉页脚，奇数页和偶数页的页眉不同，奇数页是章节的题目，偶数页是学校等信息，页脚是页码，如果直接设置可能会出现很多意想不到的问题。比如你勾选奇偶页不同，那么页脚会单单只显示奇数页码或者仅仅显示偶数页码，再或者你不勾选奇偶页不同，那么页眉全部显示一样的，而这些都不是我们希望看到的。

1. 如何设置奇偶页不同的页眉？

1.插入页眉（双击奇数页页眉），勾选奇偶页不同，输入对应章节标题，单击下一节，跳转到偶数页；

2.输入偶数页页眉，选择链接到前一条页眉，按钮变亮，选择下一节；

3.然后重复操作1设置后面的章节的页眉。
说明： 奇数页不要选择链接到前一条页眉，偶数页一定要选择。

2. 如何删除页眉上面多余的线？

有时候页眉设置错了，需要删除页眉进行重新设置，但是页眉上多出一条线无法删除，怎么办呢？这其实是格式的问题。具体的操作如下：

1.首先激活页眉处于编辑状态，然后在开始，样式那里选择“清除格式”。

3. 如何设置页码全部显示？

1.在第1章的第1页（奇数页）页脚处插入页码；

2.在第2页（偶数页）再次插入页眉，这样做就是不改变页眉的情况下全部显示页码。

四、如何生成目录以及如何修改默认目录设置？

1. 如何设置标题？

设置好标题就是为了后面一次性生成目录而准备的，所以最好在开始的时候就设置好。一般设置三级标题即可。

  1. 标题1设置，中文黑体，英文Times New Roman，三号，常规，居中；
  2. 标题2设置，中文黑体，英文Times New Roman，四号，常规，两端对齐；
  3. 标题3设置，中文黑体，英文Times New Roman，小四号，常规，两端对齐；
  4. 注意，不要勾选“全部大写字母”和“小型大写字母”。

2. 如何修改默认的目录设置？

默认的目录通常不符合毕业论文的要求。比如说，自己在生成目录的时候遇到了一二级目录化学式字体格式不正确的问题，自己单把自己修改过来再更新目录发现又便会原来的样子，后来发现，其实就是默认的目录设置的问题。所以有必要修改成自己想要的格式。比如字体、间距、样式等等。那么如何修改呢？

1.选择引用，目录，插入目录，打开目录对话框；

2.勾选其中的显示页码、页码右对齐和使用超链接而不使用页码 （就是可以直接跳转到相关的章节），选择来自模板，如果选择正式的话没有办法修改，选择修改；

3.打开修改样式对话框，分别对目录的样式进行修改；

4.选择1.5倍间距，修改相应的字体即可，其他级目录类似；

5.设置好后，替换所选目录。

第一章，意志力不只是个传说

  1. 为什么工作压力最大时婚姻往往出现问题：人们在工作上用完了所有的意志力。当意志都耗费在了办公室，家庭就会遭殃。
  2. “自我耗损”，鲍曼斯特，描述人们对自己的思维、感受和行为的调节能力减弱的过程。人们有时可以克服心理疲劳，但是鲍曼斯特发现，如果他们因为运用意志力（或者因为做决策）用完了能量， 那么他们最终会屈服。
  3. 大脑有个系统：冲突监控系统或说错误检测系统
  4. 自我损耗对行为确实有着强大影响，但是对主观感受的影响要弱很多。处于耗损状态的人更容易感到疲倦、产生消极情绪，但感觉上与非处于损耗状态的人差别也不大。
  5. 损耗了的人尽管（又一次）没有表现出任何预警情绪，但是对各种事情的反应确实更强烈。损耗了的人与没损耗的人相比，同样的悲剧前者看了更悲伤，同样的愉快图片前者看了更快乐，同样的恐怖图片前者看了更害怕、更不安，同样的冰水前者摸起来觉得更刺骨。变强了的不仅有感受，还有欲望。吃了一块曲奇后，前者更渴望再吃一块―如果想吃多少就吃多少的话， 前者确实吃得更多。看到包装成礼品的包裹，前者更强烈地想打开包裹。
  6. 所以，如果你想要某些预警迹象，那么不要单看一种表现，而是要看所有感受是否都变强了。如果你发现一些不是特别郁闷的事情让你特别郁闷，或者一些不是特别悲伤的想法让你特别悲伤，或者一些不是特别愉快的消息让你特别愉快，那么这也许是因为你的大脑回路对情绪的控制不如平常好。
  7. 自我耗损造成的影响是双重的：一方面意志力减弱了，另一方面渴望变强了。
  8. “按时交作业”和“穿干净袜子”二者都是有责任心的表现，但是不是两者之间存在正关系呢？恰恰相反，二者之间存在显著地负关系。即“学生妖魔按时做家庭作业，要么每天换袜子，但是不会既按时完成家庭作业又每天换袜子。”
  9. 类似的情况发生在考试季节，学生的自制力消耗过大，各种好习惯都被抛弃了。临近期末考试时，受调查者再个人卫生的各个方面都有松懈。刷牙洁齿不轻快了，头发脏了也懒得洗，胡子长了懒得刮。而且身上穿的袜子确实很脏，衣服也是。
  10. 所有的这一切仅仅反应了成大事不拘小节吗？他们是明智地节省时间用来学习吗？并非如此。他们一定运用了很多意志力来让自己更加刻苦地学习，但是他们最后反而学得更少了。同样，他们报告说，在考试季节，睡过头、乱花钱这两种行为都有所增加。在考试季节，他们的脾气也变坏了，更容易动怒或沮丧。他们也许说，那段时间之所以脾气大是因为考试压力大，因为大家普遍认为那些情绪是压力引起的。然而，压力真正做的是损耗意志力，而这会减弱情绪控制能力。

  11. 意志力的运用分为四大类

    • 第一大类，控制思维，学会保持关注，特别是在动机最强的时候。为了保存意志力，人们经常不会追求最全或最好的答案，而是追求实现就有的结论。
    • 第二大类，控制情绪，专门针对心情时，被心理学家成为“情感调节”。最常见的是，我们努力摆脱坏心情或是不愉快的念头。情绪特别难控制，因为你一般不能运用意志力改变心情。你可以改变你的想法或行为，但是你无法强迫自己高兴。人们运用间接策略赶走悲伤或愤怒。比如，努力想其他的事情转移注意力，或者去健身房锻炼，或是沉思。
    • 第三大类，控制冲动，大多数人一说到意志力，就会想到这个：抵制诱惑的能力，如烟酒等等。严格来说，“控制冲动”这个词用词不当。你控制的其实不是冲动而是反应 。
    • 第四大类，就是研究者所说的“控制表现、绩效、成绩”，把能量集中用于当前任务，既要达到一定的速度又要达到一定的准确度，在想放弃的时候坚持下去。

  12. 一次只做一件事。如果你一心多用，你也许暂时可以兼顾一下，但是，随着意志力消耗殆尽，你越来越可能犯严重错误。和阿曼达·帕默尔所采取的方式一致。

  13. 清单上的项目太多了，没有一个人有那么多的意志力同时做到。如果你要开始一项新的体力锻炼，就不要同时控制开支。
  14. 你只有一条意志力供应线，“新年任务“清单上的各项活动相互竞争。要每次只做一件事，而减少在其他事情上的投入。

第二章，驱动意志力的能量来自哪里？

  1. 病人为什么嗜睡？身体把所有能用的能量都用于对抗疾病，匀不出多少能量来锻炼、做爱或吵架。甚至不能进行多少思考，这个活动需要大量血糖。葡萄糖本身并不进入大脑，而是转化成神经递质，神经递质是传递信号的化学物质。如果你用完了神经递质，你就会停止思考。
  2. 研究者发现，低血糖患者与一般人相比，更难集中注意力、更难控制负面情绪。总的来说，他们比一般人更焦虑、更不高兴。
  3. 没有血糖就没有意志力。
  4. 日常生活中，对自制力的要求越高，对甜食的渴望就越强烈。
  5. PSM，月经周期以排卵日为界分为排卵前的滤泡期和排卵后的黄体期。在黄体期，女性身体开始把大量能量输送给卵巢供其相关活动，比如，产生大量荷尔蒙。但是大多数女性在黄体期多摄入的卡路里任然不够多。如果能量不足，身体就会实施定额分配，生殖系统优先，其他部位获得的能量就不够。研究一再显示，女性失控的可能性低于男性，但是在黄体期一失控就非常严重。
  6. 喂饱恶魔，葡萄糖的消耗可能会让最迷人的同伴变成恶魔。有句话说“早餐要吃好”，其实是整天都要吃好，特别是在那些面临巨大身体或心理压力的日子。如果你要考试，或者参加一个重要会议，或者完成一个重要项目，那么不要空着肚子。还没吃晚饭，就不要与伴侣讨论严肃话题。

第八章，超能量帮克来普顿和卡尔戒酒

  1. 双曲贴现（hyperbolic discounting）
    双曲贴现的结果是，当诱惑还很遥远，我们可以忽略诱惑；但是，当诱惑就在眼前，我们就会头脑发昏，忘掉长远目标。
    例如，如果你赢了彩票，可以选择6年后获得100美元或者9年后获得200美元，你会怎么选择？大部分人会选择200美元。但是，如果选项变成今天获得100美元或者3年后获得200美元，你会怎么选择？如果是理性折现，那么人们会再次应用同样的逻辑得到同样的结论，即为多得 100美元多等3年是值得的。但是，实际上，大部分人这次选择了今天获得 100美元。立即兑现这一诱惑强烈扭曲了我们的判断，所以我们非理性地贬低未来奖励的价值。安斯利发现，随着我们距离短期诱惑越来越近，我们折现未来的倾向沿着双曲线滑动；正是因为如此，所以这个缺点叫做双曲线贴现。
  2. 戒除烟酒的一个办法是回避酒吧，但是不总是奏效。还可以借助“明线”(bright lines)的帮助。明线规则，是明确、清晰、简单的规则。越过明线时，你一定会注 意到自己越过了明线。如果你答应自己“适量”饮酒或抽烟，那不是一条明线。它是一条模糊的边界，没有明确指出你在哪个点从“适量”变成了“过量”。边界那么模糊，而你的大脑又那么擅长忽视小错，结果，你也许在不知不觉的情况下越界太远。所以，你无法确信你总在遵守适度饮酒的规则。相反，零容忍是一条明线：完全戒除，任何时候都没例外。它并非适用于所有自我控制的问题——节食者不可能一点食物都不吃，但是它在很多情况下行得通。

一、珍惜研究生涯，付诸行动

带着迷惘来，不知自己想要什么，不清楚自己又要去往哪里，未来何去何从，也没有答案。在不知所措中任由时间流逝……不知这是不是当下的你？

而那千真万确的是曾经的自己。不知所措让自己失去了很多的时间和机会……

三年的时光看似很长，其实很短。大约两年前，当自己再次回到这一座久别的城市，像一个老朋友再聚首，深感时光飞逝，恰如《汉中》那首歌里唱到的，“很多事情都变了”。

暗暗告诉自己，再次离开的时候，一定要变得不一样。

不知道做点什么，但听说多看点书总是好的，于是拼命阅读；阅读没有效果，记不住书中的东西，但听说写作可以帮助自己更好的理解和记忆，所以开始尝试写作；写着写着发现没有东西可以写，但听说大量输入才能有大量输出，所以又坚持阅读……周而复始！

再回首，原来自己已经在路上了！

你看到了什么了吗？行动！你是否在期待一个梦幻般的开局，迟迟不付出行动？是否无所事事选择睡觉上网来填补？是否担忧未来何去何从而惶惶不可终日？你担忧的是什么？迷惘的又是什么？没有人知道，只有你自己。

担忧只会浪费你的时间，没有任何的益处。不如行动起来！只有开始去做点什么，你才能看到你缺少的是什么。一个人只有在做事的时候才能暴露自己的问题！只有看到自己的问题，你才有可能解决它，让自己变得更加的优秀。不用再问别人，这方法有用吗？这样做值得吗？我到底要不要……不用纠结这些问题，大胆去尝试就好了，碰壁了就碰壁了，又能怎样呢？一生，其实就是在不断的试错，在试错与实践中学习和成长！一犹豫一纠结，时间就过去了，机会就不再来了。

让自己变成一个有兴趣的人，对新鲜事物怀有一份好奇。要知道，当你对一些事情失去兴趣的时候，你不再是一个有趣的人，重要的是，可能你也会与好多的东西擦肩而过。比如，别人给你说什么或推荐什么的时候，你只回应“哦”的一声，就再也没有下文，不去尝试，没有兴趣，你就失去了一次认识和了解的机会，并且以后还会失去的更多。

很多的事情可以去做。阅读，可以让你的心灵变得富足，而精神世界不再是一片荒芜；写作，引发你的思考，表达自己的想法；实践，验证你看到的听到的，更可以认识和了解自己；TED，浓缩的精华，字字珠玑，扩大自己的眼界 ；还有公开课，缺什么提高什么……这些都是有意义的事情，慢慢的用它们来填充自己，不再犹豫和担忧，只需行动，后面的你自然就知道了答案！关于兼职和社团等活动，体验一下就好。如果占用太多的时间，不如放弃，以后有的是机会挣回来。

而自己，如今看来，想变得不一样的想法，的确是高估了自己……

但那又怎么样呢？心系远方，即使到不了又有什么遗憾？年轻难道不就是这样吗？不断的摸索，不断的受挫，再次不断的摸索……

没有什么成就，更没有遗憾！这是自己这两年的写照。问题是：离开这里后，你会因没有珍惜研究生涯而遗憾吗？

二、提高效率，享受生活

这个标题是系列文章的核心：帮助同学、师弟师妹提高自己的效率，快速成长起来，然后尽可能的享受自己的生活。

或许有人想问，如何提高效率？

先把上面的问题放在一边，想想，你需要提高自己的效率吗？也许你的效率更高，那就没有需要了对吗？大可略过这里就可以了。如果你的答案是确实有需要，再往后看也是不迟的，但是往后看也无法保证能帮助你提高效率，只有个人想法和思路，还需要你自己的尝试和不断实践的。

如何提高效率这个问题太大了，问题一大，谁都不知道怎么做，因为不够明确啊。可以把这样的问题细化具体下去，看看自己是想提高哪方面的效率。这个系列的文章也只是有可能帮助同学们提高撰写毕业论文的效率，它本来是一个很小的一个点，但如你们看到的，由这一个点延伸出来这么多的东西，甚至还不完全的，包括那些辅助性的简单的技巧，可能有用的方法论等等。

想要说明的是，如果你确实想提高自己的效率的需要，那就想想，自己想提高哪方面的效率？怎么做可以提高自己这方面的效率？自己现在做的是否有利于提高这方面的效率？如果这些问题于你来说比较难，那就想想自己做什么事情比较慢花费的时间较多？努力缩短做这样事情时间。这并不是减少做这些事情的时间。提高效率的前提是保证质量，也就是说，在保证质量的情况下，尽可能快的把事情做完。如果这个问题依旧不知答案的话，不要紧，还记得上面谈到的吗？行动！从更小的事情上面入手，然后一点点提高效率。

以提高撰写论文效率这个话题为例，可以从最简单的小技巧开始，学习一些快捷键的使用，一些软件的操作等等。要知道使用快捷键肯定比用鼠标快捷的，使用Everything肯定是比你打开资源管理器再打开文件夹找文件效率的。关于哪些软件可以提升效率，有兴趣可以移步到自己的这一篇看看软件和工具，能提升的不仅仅是效率，更是思维。甚至，大家都会使用QQ聊天，有没有想过：QQ聊天是可以不用鼠标的。 （按下快捷键Ctrl+Alt+Z，想和谁聊天直接输入他的名字，聊天窗口就打开了）这些小小的技巧，都可以节省你的一些时间，看似每次节省不了多少，但累计下来其实能节省很多的时间。还有，输入法，你的打字的速度快吗？提高输入的速度节省的时间更是可观的。双拼、五笔，花一点时间进行刻意练习，肯定是值得的。

再想提高层次的话，学习一点时间管理的技巧吧。诸如，减少不必要的事情，多用用批量处理，多使用要事优先原则等等。

提高效率的目的不是耍一些看似比较酷的招数，而是，把自己从繁重的体力劳动从拯救出来，节省出时间和精力，然后尽可能的享受自己的生活。

三、主动发现和解决问题

你善于发现问题和解决问题吗？如果不知道，那就看看身边有没有善于发现问题和想解决问题的同学，看一看他们是怎么做的。和他们对比一下，看一看差距在哪里。

当一个 人说他没有问题的时候，有可能他是没有发现问题，而不是真的没有问题。

想要提高自己解决问题的能力吗？很遗憾，自己也是在摸索的路上，所以给予不了 太多的指导。但是有一点值得与你分享：想要解决问题，就首先要弄明白自己所面临的问题是什么。这个是必须要弄明白的问题，弄不明白自己面临的问题，就更谈不上解决问题。所以首先是提高自己发现问题的能力。可以考虑采用不断深入提问的方式去发现问题，让问题一点点的浮现出来，这是第一步。另外可以参考《你的灯亮着吗？》这一本书的逻辑，让自己开始主动发现问题。

  1. 问题是什么？
  2. 这个问题是什么？
  3. 什么是真正的问题？
  4. 这是谁的问题？
  5. 问题是从哪儿来的？
  6. 我们真的想解决问题吗？

主动发现问题和解决问题是一个需要不断尝试和实践的过程，并且是一种可以通过不断思考和实践提高的能力。

做科研也好，工作也好，生活也好，自己看来，就是一个不断解决问题的过程。如果有人替你解决了所有的问题，那么就没有必要自己去解决问题的对吗？假如别人真的替你解决了所有的问题，你应该会对他产生依赖吧。那么，如果有一天他抽身离开，你能想象自己的世界将会是怎样的吗？

重要的是，这样的情况可能多吗？不，少之又少！总有些时候，总有些事情，你需要一个人去面对。如同著名台湾女作家龙应台在《目送》中，《山路》里形容病床上的才子沈君山的那一段话：

有些事，只能一个人做。有些关，只能一个人过。有些路啊，只能一个人走。

四、一件事情的毕业，永远是另一件事情的开启

路上有很多陷进，前面是一片的黑暗。曾经，没有人告诉自己，这里有个陷进，不绕过去就会跌倒，然后自己真的就跌倒了；没有人告诉自己，前面有一堵墙，不停下来就会撞到，然后真的就撞到了；没有人告诉自己，那里是一个死胡同，不退回去就没有出路，然后真的就堵死在胡同里了。这一路走的跌跌撞撞的，付出沉重代价之后，终究渐入正轨。

自己经历的，陷进也好，墙也好，胡同也好，尽自己拙劣的绘画技能，把它们一一绘制出来，于是有了这一幅并不十分好看的，又不尽完整的地图。以师兄的名义，尽可能美的呈现在你们的面前。仅仅，想尽己所能，引导你们走一段。 不要在师兄曾经跌倒过的地方跌倒了，站在师兄的肩膀上，避开那些本不该掉下去的陷进。

师兄做不了摆渡人，无法渡你们到河的对岸。那就做一回掌灯人，为你们点亮前进路上的一盏灯。自己的作用也仅是如此。

后面的路需要你们自己一步一步走，因为没有人知道你们各自想要去往哪里，只有你们自己！

一路的陪伴，如同，一场欢愉的宴会，终究会曲终人散！这个系列的文也是如此。从最简单的技巧，给予细微的指导，到试着放开，鼓励你们自己去尝试，再到暖暖的忠告，它的生命周期也仅限于此。而于你们，如同那本书的名字，《最好的年龄才刚刚开始》。

通常，故事的最后，总要以华丽的辞藻、振奋人心的言语收尾的。但，似乎不重要了，因为你们可能已经在路上了，不需要再有人替你们呐喊助威。所以，自己想做的仅仅是，跳出以往的陈规，送上自己仅有的祝福：愿你们都能走的更远，最终达到你们想去的地方！

一、如何快速入门研究领域？

这个题目挺吸引人的估计，符合现代快节奏的要求啊！如今的社会都普遍比较倾向于多快好省。

不过可能接下来的这些东西会让你失望，因为这里真的没有像前面的那些小技巧来帮助你实现那样的想法。这里能给予的是一种思路，一种解决问题的思路。但是它需要你付出自己的努力，没有太多捷径的不懈努力。

回到主题，如何快速入门一个领域呢？这个部分受启于《如何快速入门一个陌生知识领域》，Heidi格物志和《做一件事如何突破「擅长」达到「精通」》，萧秋水这两篇文章，（有兴趣的同学可以移步过去看看）然后结合自己的实践经历，总结出以下四个步骤：

1.大量阅读、泛读，记录专业词汇及术语

一般而言，在入校之后，每个人都会确定一个培养计划。培养计划里面就有自己的研究方向，然后根据自己的研究方向，提取出自己研究方向里面的核心关键词，再由自己的关键词进行展开，大量搜集资料进行泛读。

大量阅读过程不要求记忆和全部理解，只是要保证一定的阅读量。目的是对自己的领域有一个大致上的 了解。

进行大量的阅读和泛读，记录专业词汇（包括中英文）和概念。不要求弄明白每一个专业词汇的意思，也不要做任何的评判，以开放的态度去接纳你所看到的。因为有些文献之间会存在一定的矛盾，如果在泛读的阶段去做评判可能会让自己变得混乱，同时也可能降低自己的效率。所以只管去接纳就好了，等自己有足够的了解认识之后再去辩驳也不迟的。

2.找出专业词汇之间的联系

这是一个整理的过程。把前面记录的专业词汇做一个梳理，看一看这些专业词汇之间存在怎样的关系：隶属关系？因果关系？相近关系？鉴于大多数的专业词汇属于隶属层级的关系，所以整理的工具选用思维导图就挺合适的。可以一目了然地看清自己所研究领域的大致全貌。下面是自己曾经整理的一个专业词汇的例子。

从思维导图的中心的关键词出发，逐层逐级的延伸下去，很容易记住这些专业词汇，清楚彼此之间的关系，所以不至于混乱。

3.依据关键词进行主题阅读

严格来讲，这里的主题阅读并不是真正意义上的主题阅读，而是针对于某一个研究的问题进行阅读、精读。

和前面的大量阅读、泛读不同的是，这里要开始深入下去，不再是不求甚解，而是需要知其然还知其所以然。

做研究工作真正的任务是：发现问题，找到解决问题的办法，并证明这样的解决办法是行之有效的！

针对某一主题，如何进行精读呢？个人认为要注意以下四点：

  • 首先，看清这一领域的问题是什么，这一点通常会在文献的引言部分明确提出，说明当前存在的问题；
  • 其次，要了解为了解决这样的问题，研究者提出怎样的解决思路，以及采取什么样的实验方法进行研究。
  • 第三，看看实验的结果如何。研究者会做出相应的结果描述（文献阅读到了一定量之后，可以考虑自己描述一下实验的结果，为后面自己写作做准备）；
  • 第四，重点啊，分析环节。针对这样的实验结果，研究者会做出怎样的解释呢？一定要弄明白研究者是如何分析的！尽可能用自己的话把他分析的内容讲出来，能讲出来，说明你真的看懂了。如果说不清楚，那就再仔细体会，直到能用自己的话转述出来为止。如果你的能力不错，再想想这样的问题：他的分析真的合理吗？有没有更好一点的解释呢？

4.对主题阅读内容进行总结

精读之后，你的大脑里就应该有这样的一些知识：研究领域里面都有哪些问题，针对这一问题学者们都提出了怎样的解决办法，他们解决的效果如何，以及后面是如何分析解释的。这些都是非常有价值的信息，尤其是分析和现象解释的部分，因为好的分析会让你的论文提升一个档次的。但如果不进行总结记录，很可能隔两天就忘记了，甚至睡一觉起来就不记得了，所以一定要及时把这样的东西总结记录下来。记录的工具可以是Word也可以是为知印象这样的云笔记软件。

二、英文文献读不下去怎么办？

这个部分是针对一些看不下去的或者害怕看不懂的同学，如果你不存在这样的问题的话，可以直接跳过这里。

记得前些日子自己发表了一个心情（通常都很少发），记录了一点自己的感悟，直接复制过来：

那些你曾经恐惧的，不敢面对的，让你痛苦的，甚至有意无意逃避的，当你正视它的时候，它早已经退缩回到了自己的壳里。恐惧也好，痛苦也罢，它本身不可怕，可怕的是你迈不出去面对它的那一步！你有意无意忽视的问题或者当做问题根本不存在，但它不会因为你的忽视而自动消失，它就在那里，不面对不解决，它就始终会挡在你前行的道路上。

2.开始阅读，查找记录陌生词汇

当你突破心理障碍了，那又什么好犹豫的呢？开始阅读啊！

怎么阅读？是打印出来阅读还是阅读电子版的？其实都可以的。不过可以尝试这样的方法：前期阅读打印版的，后面大量阅读电子版的。

可以参考一下《获取文献原文的7种途径》，左思这篇总结，已经比较全了。当然自己并没有通过他总结的这样的方式获取文献，原因不是他总结的不好，而是，自己有同学在其他的高校，需要什么文献直接找他帮忙就可以了，完全能够满足自己的需求。如果你们也有这样的机会，那么一定要利用好。如果没有的话，可以考虑他里面列举的获取文献的方式，比如在小木虫论坛求助，或者使用像花刺软件那样的代理（代理非常耗费时间，慎重选择）。提到获取文献这一点，其实，是希望你们不要受到学校资源的限制，把视野打开。总之，你们要找到一个适合自己的途径，获取自己想要的文献。

2.如何管理文献？

当文献积累到了一定的量时候，你会发现，想要快速找到自己曾经看到的文献变得困难，这时候，你就要考虑管理好自己的文献了。

那么如何管理文献？

首先是需要用什么工具去管理？当然推荐使用前面提到的第三方文献管理软件了，比如自己用的EndNote。

其次是文献管理的原则：轻整理，重搜索

尽量不要使用建立大量文件夹分类的方式，一来不效率，二来对于模棱两可的文献难以归类，给自己造成困扰。顺便提一下，下载文献尽量使用PDF格式。一来CAJViewer支持PDF（可以识别文字），二来EndNote可以一次性导入，但CAJ和CAA格式的不支持。国内的CNKI E-Learning据说可以批量下载，但是几次尝试失败后，直接放弃了。下面是自己的实践：

    自己通常的做法是，把文献放置在一个文件夹里，然后一次性导入到EndNote里面。再次下载文献需要保留的就按照日期排序放进去。

  • 更改文献的名称

四、综述文献如何撰写？

1.要不要写综述文献？

看到这样的题目，先不要看如何写。想一想，有没有必要去写？为什么？

每个人都有各自不同的情况，所以自然会有不同的答案。这个其实也不是十分有必要的事情，因为撰写论文是需要耗费时间和精力。如果有更重要的事情要去做，那就把精力和时间花在最重要的事情上面就好了。这样说吧，撰写综述文献的价值是帮助自己对某一个领域进行一定的了解和总结。有助于了解该领域的过去研究情况，当前的热点，未来的趋势。仅仅是有利于帮助了解，不能保证就能了解。

国内综述性文章写的是比较的多，但质量好的比较少。主要原因是，在国内大多数撰写综述文章的人都是初入该领域的研究生，这并不是说研究生就写不好综述文献，而是由于刚入门，对该领域缺乏足够深入的认识，难以形成自己的见解，所以也把握不了未来的趋势。举例来说，不少学生仅仅看了十几篇文献就开始写了，那样首先是对于整个领域的整体不够了解，也可能存在偏信则暗的情况。有深入了解的人更重视实际的实验研究，倾向于提出独到的见解和研究方法。

在国外，撰写综述的人，大多是在相应的领域有所建树的、资格较老的研究者，他们了解研究的过去，清楚当下的热点以及未来的方向，所以他们能够总结出来一些真正有价值的东西，以及有自己独到的见解。但往往他们的精力有限，少有能抽出时间来撰写综述。

所以，如果你觉得没有必要的话是可以不写，后一部分就可以略过；另一方面，如果需要总结一下自己所研究领域的大致情况，那请就往下看如何撰写。

2.怎么撰写综述文献？

这篇文章是年前的一次总结，这一次做了适当修改。

当你看到这篇文章，说明你有撰写综述文章的需要。因此，这里给你提供一个写综述论文的整体思路，仅供参考。

下面是自己撰写综述论文所耗费的时间日志，从20号到29号，零零散散总共历时27小时27分钟40秒。可以参考一下，有个心理准备，知道大概得花费多长的时间。


经历了三大阶段：准备阶段、撰写阶段、修改阶段。其中前面两个阶段耗时最长，也最乏味，能够写完，完全是依靠完成论文有其他用途的使命感。期间也浪费了不少时间，有时精力不够集中，不知道自己究竟在干啥；有时抵不住诱惑，浏览网页耗费时间来逃避写论文的痛苦；有时由于晚上睡眠时间不够，导致白天太累不能完成。总的来说，也算马马虎虎的完成了。阿Q一下，估计很多同学也跟自己一样，不知不觉的浪费了许多宝贵的时间吧，所以这么想心理也得到了些许安慰。

那是不是写完论文就完成了呢？

当然不是，最重要的环节来了——总结。如果不回顾总结，那就不可能得到收获，看似总结浪费时间，其实却是学习收获最多的环节。所谓慢慢来，比较快。废话不说了，以下将从这三个阶段分别做以梳理，回顾下这篇综述文章究竟是如何完成的，以及还有哪些地方需要注意的，下次该如何修改。

准备阶段

准备阶段准备点啥呢？个人认为有三个地方值得注意：

（1）理解题目，确定关键词

适合于题目已经拟定的综述类情况，比如给定的综述论文题目。理解题目就是说，你要清楚题目是让你写的什么内容，找出和题目相关的关键词，其他给定的题目也可参照如此，怎么确定关键词就不用说了。综述类论文还有一种情况就是没有给定题目，那就需要自己确定题目，那如何确定题目？首先要对这个领域有所了解，找到哪个还没有作者写过的内容，这需要注意确定题目不能太大或者太小，太大掌控不了，综述不能深入和全面，太小又没什么可写的，这个度不好把握。第二大类的小论文，也是写作最多的实验性的论文，这类论文是需要实验数据去支撑的，也可参考，找出关键词，如果能有小论文的实验框架自然最好，省去了不少时间。

（2）查找参考文献，归类整理

查找文献自然是根据关键词查找了，那除了知网还有哪里可以查找呢？Google scholar和百度学术是个好去处，这两个的优势在于可以插入参考文献，非常赞，当然也有Endnote这样更牛的软件，但个人还是更喜欢使用Google scholar，查找和插入文献。文献查找完了之后就需要归类整理了，那如何整理最为高效呢？使用Mindmanager吧，将所有的下载好的文献放在一个新建的文件夹里面，在文件夹里新建一个Mindmanager文件，然后将所有的文献Ctrl+C、Ctrl+V到Mindmanager里面，接下来重点来了，添加Tags，每浏览完一篇文献（主要浏览摘要，省时间）之后，要给这篇文献添加一个Tags，关键词，也可以多个（但最好不要超过三个，因为到后面不好归类），最后一步显然就是同类合并，归类整理了。如果有不知归类的文献那就先放在TEM（临时分支）里面吧。

（3）思考论文框架

文献都归类好了之后，就该考虑论文的框架的问题了，也就是说写作总体思路，按照什么样的顺序往下写，这一点我是根据题目中的关键词的顺序来判断的（仅供参考），然后根据写作结构来调整Mindmanager里面分支的顺序。这样准备阶段的工作差不多也就做完了。

撰写阶段：

撰写阶段是最头疼的阶段，常常浪费了很多时间。原因在于，不敢按照自己确定的框架去写，感觉总缺点什么，这和自己追求完整的性格有很大关系。不过最后还是解决了这个问题，如何解决的呢？直接写！你没有看错，直接写，不管语句，不管格式，不管思路，直接写！一旦你开始写了，就发现很难停下来了。也就说，恐惧的门槛迈过去了。这就好像我们冬天早晨难以起床一样，心理的畏难导致我们屡屡失败。同理啊，解决办法是：直接坐起来，趁着大脑还没有反应过来就开始穿衣服（等大脑反应过来衣服都快穿好了，那时你还会接着睡吗？）

接下来，就是按照论文的框架一路写下去了，写作过程中主要看参考文献部分，如果参考文献信息不够再来浏览全文。其次就是为了减少直接抄袭，最好还是进行同义替换，也就是要会运用不同的句子表达一个意思，当然不能替换的不是原话了，这点你在阅读文献的时候就会发现，经过二次引用和更多次引用的的原句肯定就不是原来作者表达的意思了。那个“传话游戏”就很好的证明了这点，毋庸置疑。

注意：在撰写阶段千万不要一边写一边插入参考文献，那样是最浪费时间的。自己的做法是，在需要插入参考文献的地方做一个标记，简单的标记，能够看懂，知道是哪个作者的哪篇文献即可。一路写下去，写完之后再在修改的过程中统一插入文献。

修改阶段：

经历了痛苦的准备和撰写阶段，这一阶段就轻松多了，但是仍然要注意。需要修改那些内容呢？有两点：

  1. 插入参考文献，怎么插入呢？自己是在Google scholar里面找到这篇文献，点击“引用”，将弹出来的参考文献复制下来，然后利用脚注的方式插入到文献中，如果有点小错误，立马去修改好，费不了几秒钟时间。这个步骤还没有完，因为插入文献的格式没有带英文方括号。如何批量修改以尾注插入的文献呢？只需要查找“^e”, 然后替换成“[^&]”，就一次性将方括号加好了，方便么？
  2. 阅读全文，修改语句不通的话语和格式错误进行修改，查找格式错误。

五、不想写论文怎么办？

当你有这样的困惑的时候，不用担忧，因为，你并不孤独！喜欢写论文的同学怕是不多，或可能就没有。不夸张的说，写论文是非常枯燥和繁琐的事情。而几乎没有人喜欢做枯燥乏味的事情。要知道，你不想写论文，大家也都不想写！

但论文还是要必须写的对不？所以问题来了：如何缓解这样的困扰呢？通过自己的实践摸索，以下的几个要点可以提供有效的帮助：

1.减少撰写论文的时间

可能有同学看到这样的小标题会困惑：减少撰写论文的时间行吗？写不完怎么办？不急，想想这样的问题，时间真的就和完成论文呈正比吗？当然是否定的对不对。不知道多少同学像自己一样存在拖延撰写论文的问题？大脑倾向于做有趣的简单的事情，对于难的枯燥的会拖延。认识到这一点就不难发现，很多人都会存在拖延的问题。所以勇敢的面对就好了。

当面临难以完成的任务，我们会在潜意识里告诉自己：哎呀，这部分太难写了，烦呐，先看会电视（上会网，打一把游戏）让自己放松放松，等心情好了再写吧，反正时间还很多的，不急这一会。然后，两个小时过去了……忽然又意识到渴了，去接杯水吧，接完水再回来，“咦，老同学给自己QQ发消息了，这家伙好久没有出现了，不知道他（她）近来在忙些什么，聊两句吧，反正费不了多少时间。”你这样告诉自己，又一个小时过去了……你一看时间，“哎呀，上午就剩半个小时了，今天周内，到饭点学生比较多，需要提前半个小时去吃饭呢，不然饭堂的人多的很，反正上午也写不完了，先去吃饭，下午再写吧。”吃完饭，“中午的面好咸啊，吃的胃不舒服啊，先歇一会，等自己状态好点时候再写吧，比如利用这个时间看一部电影吧，最近那个啥电影挺火的，赶紧找找看”，然后重现上午的情景……时间黑洞就是这样产生的啊！

拖延着不愿意开始写论文是因为，在潜意识里，我们认为时间还很多，不急这一时半会儿的。所以写论文的时间变得很长，但是真正写的时间很短的。想象一下这样的场景，后天一早必须要提交论文大赛作品，现在是晚上十点，资料已经准备好了，但你还没有动笔写，是不是很有压力和紧迫感？当然的，重要紧急啊！在时间紧迫的情况下，你会很快完成的，即使作品不是那么的优秀，但相信你会完成的对不？

看似我们写论文的时间很多，其实真正在写的时间很少，时间一多，我们就会容易丧失对于时间的紧迫感，所以就会拖延，而越拖延的后果就是越不想写。既然缺乏时间的紧迫感是造成的原因之一，那么解决问题的途径就可以从这一点入手：何不尝试减少写论文的时间呢？

2.任务分解，规定每天必须完成的事情

在思维导图那一篇里面提到了任务分解，分解任务的好处之一是更加有利于去执行。结合着青蛙工作法，规定自己每天必须完成的事情。
拿自己的实践为例，每天在做日计划的时候 ，必有一项是当日必须完成的论文写作中的某一部分，就是为了保证论文写作的顺利进展。平均下来，每天在论文撰写上面花费两个小时。当然中间也会拖延，也会走神，但一想到自己要在两个小时内完成这样的事情，所以再拖延也是有限度的。每天完成一点，每天保证一点，坚持下来，写完整篇的论文也就有了保证。每天只完成那么多的内容，把那部分的行动当成一只大青蛙来处理。看似很慢，其实非常的快。在预计的时间限度内，自己可以完成，就不会逼着自己去赶工，不会焦虑和担忧写不完论文了。

如果某一天自己只执行完了撰写论文的这一行动，即使接下来其他不那么重要的事情都没有完成（当然这样的情况也基本不存在），那么这一天也不会觉得虚度了。

学习和运用一些简单的时间管理技巧总是可以提升自己效率的。简单的一个规则：每天必须完成某件事情。这样的方法是每个同学都可以学会的，学会之后，当你真正的去做了，成就感会得到提升，甩掉浪费时间的内疚，甩掉一些焦虑和担忧，你的生活会变得更加的有意义。

3.一天当中什么时候写？

当然是精力最充沛的时候啊！因为精力充沛的时候，人更容易集中注意力。撰写论文需要高度集中注意力啊，那样可以保证思路清晰，语言表达顺畅。

那什么时候精力最为充沛呢？问一问你自己啊！看看你自己什么时候的状态最好，又不会被别人打扰的时候，找到那样的时间段。

通常而言，每个人的情况可能不同，但是大体上还是相近的。比如，大多数的人会在早上的时候精力比较好，比较安静，而且不会受到外在的打扰，那么就可以利用好早上的时间段开始写；中午是午休的时间段，但是此时人的状态大都不佳，可以考虑做些不那么需要集中注意力的，又可以保证论文写作顺利的事情，比如整理数据，处理一下图片等等；下午后半程和晚上的前半程人的精力也还可以，但存在经常被打扰的可能性。一旦被打扰，思路就会被打断，想要再次集中注意力就需要花费一些时间，研究显示，被打断后再次集中注意力需要15分钟时间；另外，可能有些同学觉得自己在深夜的时候精力最好，像自己之前一样，属于那种夜猫子型的，但是，最好不要那样，能不熬夜尽量不熬夜，熬夜没有一点好处。

自己通常是早上的两个小时写。安静，没有人打扰啊！有人说，冬天太冷，夏天又太困，起不来啊！这不是在给自己找借口么？其实早起没有那么痛苦！之所以自己能够保证每天早上去完成这样的事情，不是因为自己意志力多么的强大。和你们一样，自己也是经常睡到九、十点起不来的。但有一些事情要做，由不得自己啊。已经工作的同学会有体会，即使再困再冷，你都很少错过时间去上班的。当然，自己现在已经习惯了早起了，熟悉的人知道自己在执行【百天行动计划】 （早起跑步），其实就是为了保证论文的写作。在执行计划开始的一个月内，比较痛苦，从夜猫子转变不是那么的容易，慢慢的，身体就适应了，早起没有那么的痛苦了。一到点就醒，一醒来就起床，一起床就去跑步……一连贯的动作就可以保证在八点前到达实验室开始一天的工作，这是由行动计划带动写作进展。回想看，现在还是蛮享受这个过程的。

4.适当放松，不逼迫自己

身边的人看到的是自己如何如何的积极，其实，那并不是积极，而是焦虑的结果。因为焦虑，所以逼迫自己！身边的人也会经常告诉自己，不要有那么大的压力，其实不是有多么大的压力，而是自己已经快到慢不下来，不是进步的快，而是把时间过的很快，把生活的节奏过的太快了。这是一种病态。所以不希望你们进入这样的漩涡，焦虑是无底洞，深不见底……

慢慢地，不断地问自己：真的有那么重要吗？开始还是会固执的认为：是的，真的很重要。慢慢的，这样固执的想法开始动摇了，直至得到了否定的答案：其实，也没有那么的重要。所以既然没有那么重要的事情，自己何必去做无谓的担心和焦虑呢？因此也就有了如今的答案：不再逼迫自己！给放弃自律留一点点空间！

不想看文献，那就不看了，想看的时候再看！不想做实验了，那就不做了，想做的时候再做！不想写论文的时候，那就不写了，不强迫自己写了！

这不是说真的就那么的自由，而是，在自己能把控的范围内，尽可能地给予自己更多的自由。比如不想写论文就不写，是因为自己知道，自己即使这会不写，总会有时间和精力去完成这样的事情，而用不着强迫自己。

做到张弛有度需要不断实践，但是这个其实是可以习得的能力。所以于你们来说，努力去尝试就好了。

一、思维导图是什么？

在前面的几篇文章里面，已经反复提到了思维导图或者MindManager软件，那么究竟思维导图是什么？又该如何认识它？首先，想要认识一个事物，就应该从它的定义出发。百度百科上对思维导图给出了这样的解释：

思维导图（Mind Map）又叫心智图，是表达发射性思维的有效的图形思维工具，它简单却又极其有效，是一种革命性的思维工具。思维导图运用图文并重的技巧，把各级主题的关系用相互隶属与相关的层级图表现出来，把主题关键词与图像、颜色等建立记忆链接。
百科上的解释是发射性思维，这个显然是不正确的。根据思维导图的创始人托尼·巴赞在《思维导图：放射性思维》（自己很偶然在图书馆发现了这一本书，也因此和它结缘）中一书的观点，应该是放射性思维！

但是不要紧，不妨碍我们认识它！从定义上看，思维导图是一个有效的思维工具！它是用一个中心关键词以辐射线的方式向四周展开的图像，由颜色、文字、图形和线条等通过一定的层级关系连接起来。解释起来真的比较费力。一图胜千言，那就先放一张图上看看它长什么样子。

这是一张思维导图，图片来源于管理者家园的Eyong大哥《思维导图的三招十八式》一书的作品。

上面的图例都是思维导图的应用实例，大多都是手绘的作品。自己其实也是非常佩服那些手绘的高手，因为手绘更能体现思维导图的精髓，但是自己天分有限，再者从效率的角度出发，所以选择了计算机绘制的方法。有了这一张图，就能更好的理解思维导图了。

二、思维导图有什么用途？

想到一个东西就应该考虑它有什么作用。思维导图也不例外，没有应用就没有价值。思维导图究竟有什么应用呢？

在《思维导图：放射性思维》里面分别从个人、家庭、教育、商业和职业用途、甚至未来等五大方面列举了思维导图的应用，但个人不太喜欢那样的应用介绍。为什么呢？举个例子，看这篇文章的你，角色可能是学生，你还没有自己的家庭，还没有工作，那家庭、商业和职业方面的应用对你有什么作用吗？当然没有多大的应用。因此，我们可能更多的关注个人方面的应用，而忽略了一点，那就是其他方面的应用可能也有很高的参考价值，因为思维导图的精髓是发散性思维，所以它的应用也必须是用发散性的思维去应用。

三、思维导图如何运用在科研当中？

思维导图如何用在科研当中？这是个好问题！

要回答这个问题，先想想做研究、撰写毕业论文都有哪些工作要做？不动脑子地列举：开题、做实验、整理数据、撰写毕业论文，准备答辩。归并一下，从两个应用侧重说明：开题与答辩演讲思路和构建毕业论文框架。

用途一：理清开题与答辩演讲思路

开题和答辩放置在一起是因为这两者有一定的共性，首先两者重要不言而喻；其次这两者都需要做PPT给大牌们演讲。

不少同学惧怕这两个环节，因为稍有问题就会被那么多的眼睛钉在上面下不来，无地自容。不信的话，去看看你们的师兄师姐的开题与答辩就会明白。不是危言耸听，真的遇到了确实比较尴尬。问题要么是基础知识不扎实，要么是没有底气，要么就是思路不清，讲着讲着就卡壳了，甚至紧张，害怕忘记的紧张，结果往往真的就会忘记的。这些都会影响自己的信心和底气。而有没有底气，在下面的人会看的很明白。

演讲需要用到PPT，PPT的质量很大程度决定了演讲效果。

制作PPT是一门技术，除了排版、配色这些辅助的技巧外，真正保证PPT质量是里面的内容和逻辑结构。逻辑结构清晰，内容完整，基本就可以做一次很好的演讲，至少不会出太大的差错。以自己的选题报告为例说明，整个过程分为六步：

第一步，用思维导图整理出自己的框架结构，保证逻辑清晰

通常，如果是一件陌生的事情，我们很难对其有一个清晰的认识。开题报告就是一个好的例子，如果不用思维导图去整理清楚自己的思路，那么脑海里必然是浆糊， 演讲的结果也是混乱的。

演讲的核心是清晰的逻辑，这里就是开题的主线。没有主线，很难说的上是好的演讲。就如人一样，有自己的骨架。没有骨架直立都不可能实现的吧。所以，一定要理清自己的思路，理清演讲的主线。下面是自己的一个范例，展开了选题依据的子内容。

第二步，依据自己的框架结构制作PPT，增加减少内容

理清了自己的思路，就有了演讲的框架，制作PPT就可以依据这样的框架去补充完善，PPT的章节和框架保持一致。这样做的好处有二：一是不至于太过凌乱；二是便于记忆。对应的PPT的主题就和思维导图的保持一致。接下来讲什么，怎么讲，一级一级深入下去，逻辑就清晰了。

另外在做PPT的时候，有几个地方需要注意：

（1）重点一定要突出

泛泛的讲演对于听者来说是一件痛苦的事情。人的记忆力和注意力都十分有限，如果PPT没有重点，很难给听者留下好的印象。

（2）文字要少，精炼；字体要大，醒目

有的同学会在PPT里面添加好多的文字，结果字数太多放不下，怎么办呢？就选择把字体调小，然后照着上面的逐字逐句的念。试问，假如你是听者，你会是怎样的感受呢？还有，下面坐的都是年长的人啊，视力没有那么好，把字体调小，他们能看清吗？引用一下一位老师当场说的话，“我不需要你来给我做科普！”你能听出话里的不耐烦和不满吗？所以一定要考虑听者的感受。少用点文字，尽可能精炼，还有，把字体调大一点，让他们能够看得清。

（3）不要做的太花哨，三种色彩最适宜

提及这一点有没有必要呢？当然有啊！开题和最后的答辩都是严肃的场合啊，把PPT做的那么花哨真的就好看吗？就能给自己加分吗？懂一点配色还好，如果不懂配色，会弄巧成拙的！再换一个角度，时间有限，在制作样式上花费较多的时间，那么在整体内容上花费的时间就少了。能不能给出不错的内容就无法得到保障。逻辑结构是核心，内容是灵魂啊！一般而言，三种颜色足以，简洁大方，最多不要超过五种颜色！

第三步，反复检查，不犯常识性错误

这一点重要性也不必说的吧。任何的小错误出现在演讲的PPT里面都可能让听者反感的。比如，千万不要出现错别字、语句不通、标点符号运用明显错误的情况。任何一个小的错误都会直接导致听者对你的印象大打折扣！

第四步，思考“他们可能会提哪些问题？”以及“自己该如何回答？”

在演讲后会有几分钟的提问环节，导师们会针对你的演讲提出一些可能存在的问题。诸如，“你为什么选择这种材料？XX不是更合适吗？”“你说到这两者有较大的相似性，是你自己想的还是有其他参考依据呢？都有哪些相似性？”……

想一想，他们可能会提出什么问题，列举出来，然后针对这样的问题做出相应的解答。要知道，只有准备充分了，才不会显得没有底气和信心。

第五步，预演，把控时间和节奏

选题也好，答辩也好，都是有时间限制的。不可能任由某一个人在摆龙门阵，漫无边际的大侃特吹，说太多是体现不出重点的；也不可能允许某一个人说个一两分钟就结束，时间太少是说不明白的。之前一位同学在参加创新项目申请的时候，没有把握好自己的时间，时间还没有到，自己已经把内容讲完了，结果与项目无缘，甚是可惜。所以需要根据自己的演讲内容把控好自己的时间和节奏。在独处的时候，定一个闹钟，多预演几次。

第六步，演讲，声音洪亮，自信

这一步注意一点：演讲的时候提高自己的声音。原因是，提高自己的声音可以克服紧张情绪，给自己增加演讲的底气。这一点是自己导师教会自己的，很受用。

毫无疑问，所有的准备都是为了最后时刻的精彩发挥！没有前面那么多的精心准备，想要获得较好的演讲效果，恐怕说出去自己都是无法相信的。如同任何一个考试一样，没有充足的准备，即使过了十有八九也是运气使然，不见得以后的路都会如此的顺利，总会有跌到的时候。

用途二：搭建论文的框架与任务安排

思维导图强大之处在于，理清思路与逻辑，按照一定的层级关系呈现出来，便于站在更高的层面去看清一些事物。都说站的更高，看的更远。这话确实不假。因为俯视，真的可以看清很多的东西。

既然思维导图可以运用到开题或答辩的演讲当中，那么，对于同样需要清晰结构的毕业论文，应该可以发挥更大的优势吧。看多了学位论文的同学会发现，大多数的学位论文的逻辑是比较清晰的，这一点看目录就可以清楚。但，在非正式的场合，比起目录，思维导图应该更能让人记住自己的思路和逻辑。这和人的大脑有关，发散性思考是人脑的自然思考的方式，以此方式来感觉、认知和记忆我们见识到的事物。以图文并茂、层级结构呈现，可以充分利用左右脑的机能来进行认知。以自己的论文框架为例，下来说说，怎么运用思维导图。

先给出自己的图例。有人会说，看不清什么啊！是的，真实的东西不能让你们看清的。不过不要紧，你们知道它长什么样子就好了。

结构清晰吗？一篇论文都有哪些内容，哪一部分写些什么，怎么写，都呈现在自己的眼前。这是自己的，所以自己就能够统揽论文的大局。所谓运筹帷幄决胜千里，自己看来，就是站在更高的战略的层面去看待整个局势。

问题来了：怎么做呢？又有什么用呢？下面依次说明这些问题。

1.怎么构建论文框架呢？

（1）构建大致框架

构建框架的办法前面已经说到了，参考啊！参考其他学位论文的结构，参考师兄师姐的结构！看清一点，是大致框架！千万不要指望一次构建一个完美的框架。要知道，罗马不是一天建成的。

（2）修改完善内容

有了大致的框架就可以进行适当的调整，增减必要的内容，不断完善这个整体。

这么做有什么好处呢？举个自己的例子，在整理的过程中，有一块模棱两可的内容，自己不确定放在哪里合适，其他人和自己的研究方向又不同，怎么办呢？所以，就拿这个问题请教了自己的导师，最终他给出了答案。拿自己的例子想要说明的是，如果没有站在更高的层面去看待自己的论文结构，这个问题就可能变得棘手。棘手在于，可能自己随便放在某一分支下，时间一长，自己忘记了，到最后别人帮你发现有问题，你再去被动的调整的话，如果时间紧就会给自己徒增焦虑。而实际这个问题是可以自己发现然后提前预防的。

2.构建框架有什么用？

不知道有多少的同学有时间管理的概念，就是项目一定要分解成任务，任务要再分解成下一步行动。而撰写毕业论文就可以看成是一个大项目。问题是，写完毕业论文这么大的项目你准备怎么完成它？

作用一、分解撰写论文到具体的行动

比如，你看到“第2章 实验方法”，你知道干嘛吗？有同学说，写第2章 实验方法啊。问题是，第2章 实验方法你怎么写？写什么东西？花多长时间写呢？你会语塞的。撰写第2章是大项目下的一个大项目啊。直接写是写不出来的，写出来的话，逻辑结构也是很难保证的啊。所以构建框架的作用之一就是便于分解项目到任务，分解任务到下一步行动。

作用二、跟踪撰写论文的过程，保证写作顺利进展

拿实验设备为例，比如自己想要写实验设备这部分内容，那怎么写呢？光看着“实验设备”这几个字你很难知道怎么做，所以分解啊。比如自己分解的两个行动：记录，实验设备的名称、型号等相关信息；制作表格，实验设备列表。以动词开头，这两个行动就具有可执行性，看到这两项行动，就知道，哦，要做什么什么 。同样的，想写拉伸实验部分，再做分解，分解成具体可以执行的行动，该查找资料的就查找资料，该写作的写作，该绘制CAD图的就去绘图，一项一项去执行。前面的小圈圈就代表了完成的情况，哪些已经完成了，哪些还没有，一目了然。如此一来，就可以站在更高的层面清晰地跟踪论文写作的整个过程。

四、思维导图和你有什么关系？

不止向身边的一两个人推荐过思维导图工具。但慢慢地发现：原来大家都不需要。通常给大家推荐这一工具，如石牛沉入大海，换回“哦”的一声，或者“好的，有空看一下”，然后就再闻不见其他的声响，也再也没有空看了，然后什么也没有发生。

如同这个系列的文章，不学习，不交流，不使用，它对你就没有多大的用处。也就是和你们没有关系。

渐渐地，自己不再主动向其他人推荐什么。因为慢慢意识到，他们暂时不需要这些东西，如果需要的话，他们会自己去了解，而用不着自己指点。所以遇到愿意交流的人就交流，不愿意交流的人自己也不会勉强，继续做好自己的事情。这些东西，即使不学不用，也不会损失什么的。因为在一定的范围内，一些东西是可以被替代的。比如批量处理的，一个个处理也可以啊；比如理清思路的，不用思维导图，写下来整理也可以的；再比如不用搜狗的自定义输入，一个个敲入也可以完成的对不？所以有需求才有价值。如果没有需求，东西再好，工具再高效，也和自己没有关系，对吗？
